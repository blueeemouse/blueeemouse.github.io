Posted on 2025-09-05 Edited on 2025-09-06

感觉学到了什么呢，读这几篇关于memory management的论文？它们的主线都是什么（也就是说，它们都在解决什么问题？它们有了什么样的发展？）

另外，关于快速读论文，也稍微学到了一点。不需要每篇都读那么细致。是不是好论文，让ai给你总结清楚，一下就能看出来了；再稍微扫两眼method，就能进一步确定了。如果只是一般般的论文，也就没必要写详细的笔记了，写个简介，大概介绍一下它的方法如何解决问题，以及创新性何在即可
