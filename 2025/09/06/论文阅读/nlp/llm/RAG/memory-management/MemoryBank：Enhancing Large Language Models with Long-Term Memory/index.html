MemoryBank：Enhancing Large Language Models with Long-Term Memory

Posted on 2025-09-06 In 论文阅读 , nlp , llm , RAG , memory management

Venue：AAAI 24

date：2024-03-24

动机：

论文着重解决llm缺失长期记忆机制的问题。缺乏长期记忆，则在一些需要持续交互的场景里llm可能表现不够好

简介：

论文提出memory bank，用于存储长期记忆。且memory bank中有层级结构（即，既有每日事件摘要，也有全局摘要；同时会根据对话历史构建用户画像）
而检索记忆的时候，除了利用当前上下文，还会利用用户画像（相当于利用了一些历史数据）
记忆更新方面，论文借鉴Ebbinghaus遗忘曲线理论，构建了一个离散的、刻画记忆被检索概率的模型

insight：

memory bank，层级结构，感觉这些是比较稀松平常的。值得注意的小细节是保留记忆时带上了时间戳，这可能便利了某些问题的回答。另外，构建用户画像的操作可能也比较少见

借鉴Ebbinghaus遗忘曲线理论，实在有点像讲故事。

contribution：

提出了一套长期记忆的机制，可以让llm存储、更新、检索记忆，并支持刻画用户画像
