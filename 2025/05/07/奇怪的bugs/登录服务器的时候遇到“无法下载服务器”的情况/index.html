登录服务器的时候遇到“无法下载服务器”的情况

Posted on 2025-05-07 Edited on 2025-05-08 In bugs

问题大概就标题所述，具体的症状一般是，（之前能登陆服务器，而现在）用vscode插件登录服务器的时候，输完密码，明明没错，但就是上不去。本来输入密码是在窗口的顶上的，但在那里输入完密码之后，在终端它又要你输入一遍，完了等一会就会报错，说的大概就是这种“Failed to install the vs code server”

这个应该算是相对常见的bug吧，一般的解决方法就是，用其它方式登上服务器（例如，直接用终端，或者用MobaXterm等其它工具），进入到用户的目录下，把.vscode-server这个文件夹给删掉，之后再尝试用vscode插件登录服务器，应该就能登上了（可以参考一下这个博客）
