matplotlib画图时中文显示为乱码

Posted on 2025-06-23 In python相关bugs或知识

问题就如标题所述。最简单的方法当然是在画图里显示文字的时候用英文，就啥事没有了，绝大多数场景下也都可以。但如果就是要用中文的话，其实也可以

其实这是一个很常见的问题，之前在ubuntu上好像也遇到过。记录一下是为了在下次遇到的时候能快速解决

这个博客里的步骤还是比较详细的，我也是按照这个实操了，最后是可行的。其实比较麻烦的竟然是找SimHei字体的下载资源。这是一个可用的链接（非商用，仅用于个人学习。如有侵权之类的问题可联系删掉）（应该还有很多地方可以找到吧）

这里稍微说一下，博客里第四步是说，修改一下matplotlibrc文件，在里面加入三行。但打开那个文件，会发现全部内容都是被注释掉的，且它说是不要改动这个文件。但无需理会，这里我们有需求，所以修改是正常的。加的三行好像其实加到哪里都行。

下载完字体，且把字体放到库里之后，再在代码里加入以下两行就行：

# matplotlib画图 # 中文显示
