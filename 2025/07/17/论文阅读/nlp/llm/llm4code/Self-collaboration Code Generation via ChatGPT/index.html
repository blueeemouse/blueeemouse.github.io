Self-collaboration Code Generation via ChatGPT

Posted on 2025-07-17 In 论文阅读 , nlp , llm , llm4code

Venue：

date：

动机：

现有 LLM 在复杂代码生成任务（如多文件依赖、长逻辑任务）中表现不足，而人类通过团队协作解决复杂任务的模式可被借鉴

insight：

通过简单的“角色指令”（本质就是prompt engineering），就能让llm化作不同的角色，从而实现用一个llm就完成多个角色的分工（所以其实可以理解为，我们用不同的prompt激发了llm的不同方面的能力，从而实现了self-collaboration。self-collaboration就是说，我们完成一个任务，可能它很复杂，需要分工，有多个角色要完成各自的任务。我们可以用同一个llm，然后依次让它扮演相应角色，并给它相应的prompt）

contribution：

提出了一套框架，可以让llm扮演不同角色，处理复杂任务（感觉本质上就是一套提示词工程）

基于软件开发方法论，设计了三个角色，并组成一个虚拟团队，明确了分工和协作的规则（这里有点讲故事的嫌疑，为它们设计的三个role找到背书）
