еҺҹйўҳй“ҫжҺҘеңЁиҝҷ

йўҳзӣ®жҸҸиҝ°пјҡ

з»ҷдҪ дёҖдёӘж•ҙж•°ж•°з»„В numsВ гҖӮдёҖдёӘеӯҗж•°з»„В [numsl, numsl+1, ..., numsr-1, numsr]В зҡ„В е’Ңзҡ„з»қеҜ№еҖјВ дёәВ abs(numsl + numsl+1 + ... + numsr-1 + numsr)В гҖӮ

иҜ·дҪ жүҫеҮәВ numsВ дёӯВ е’Ңзҡ„з»қеҜ№еҖјВ жңҖеӨ§зҡ„д»»ж„Ҹеӯҗж•°з»„пјҲеҸҜиғҪдёәз©әпјүпјҢе№¶иҝ”еӣһиҜҘВ жңҖеӨ§еҖјВ гҖӮ

abs(x)В е®ҡд№үеҰӮдёӢпјҡ

  • еҰӮжһңВ xВ жҳҜиҙҹж•ҙж•°пјҢйӮЈд№ҲВ abs(x) = -xВ гҖӮ

  • еҰӮжһңВ xВ жҳҜйқһиҙҹж•ҙж•°пјҢйӮЈд№ҲВ abs(x) = xВ гҖӮ

    зӨәдҫӢ 1пјҡ

    иҫ“е…Ҙпјҡnums = [1,-3,2,3,-4]

    иҫ“еҮәпјҡ5

    и§ЈйҮҠпјҡеӯҗж•°з»„ [2,3] е’Ңзҡ„з»қеҜ№еҖјжңҖеӨ§пјҢдёә abs(2+3) = abs(5) = 5 гҖӮ

    зӨәдҫӢ 2пјҡ

    иҫ“е…Ҙпјҡnums = [2,-5,1,-4,3,-2]

    иҫ“еҮәпјҡ8

    и§ЈйҮҠпјҡеӯҗж•°з»„ [-5,1,-4] е’Ңзҡ„з»қеҜ№еҖјжңҖеӨ§пјҢдёә abs(-5+1-4) = abs(-8) = 8 гҖӮ

    жҸҗзӨәпјҡ

  • 1 <= nums.length <= 105

  • -10^4 <= nums[i] <= 10^4

    жҖқи·Ҝпјҡ

    еӣ дёәжҳҜд»»ж„Ҹеӯҗж•°з»„пјҢеҸӘиҰҒжұӮе’Ңзҡ„з»қеҜ№еҖјжңҖеӨ§пјҢеҜјиҮҙйўҳзӣ®зңӢиө·жқҘз•ҘйҡҫгҖӮеҰӮжһңжІҝз”Ёд№ӢеүҚ[[жңҖеӨ§еӯҗж•°з»„е’Ң]]йҮҢзҡ„жҖқи·ҜпјҢе®ҡд№үдёҖдёӘabs_max_fж•°з»„пјҢе…¶з¬¬iдёӘе…ғзҙ иЎЁзӨәд»ҘnumsйҮҢзҡ„з¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҢд№ӢеҗҺе…¶е®һдёҚиғҪйҖ’жҺЁгҖӮзӣҙи§үдёҠпјҢдјјд№ҺеҸҜд»Ҙиҝҷж ·йҖ’жҺЁпјҡ\[abs\_max\_f[i]=\max(abs(abs\_max\_f[i-1]+nums[i]), abs(nums[i]))\]зңӢиө·жқҘиҝҳжҢәеҜ№зҡ„гҖӮдҪҶеҸҜд»ҘдёҫеҮәеҸҚдҫӢпјҡеҰӮжһңжңүиҝҷд№ҲдёҖдёӘж•°з»„пјҡ\([7, -1, -2, -3, 15]\)пјҢйӮЈеҪ“жҲ‘д»¬иҰҒжұӮд»Ҙ15йӮЈдёӘе…ғзҙ з»“е°ҫзҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҢз”ЁдёҠйқўйӮЈдёӘйҖ’жҺЁејҸжұӮеҮәжқҘзҡ„з»“жһңжҳҜ15гҖӮеӣ дёәжӯӨеүҚжұӮеҮәзҡ„д»Ҙ-3з»“е°ҫзҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјжҳҜ6пјҲеңЁиҝҷдёӘжғ…еҶөйҮҢпјҢеҸ–еҲ°жңҖеӨ§еҖјзҡ„жғ…еҶөжҳҜд»Һ-1еҠ еҲ°-3пјүпјҢдҪҶе®ғе®һйҷ…зҡ„е’Ңзҡ„еҖјжҳҜ-6пјҢе’Ң15еҠ еҲ°дёҖиө·дјҡеҮҸе°Ҹз»қеҜ№еҖјгҖӮеҸҚиҖҢеҰӮжһңжҠҠ7еӣҠжӢ¬иҝӣеҺ»пјҢйӮЈд№Ҳ7еҠ еҲ°-3йӮЈдёӘдҪҚзҪ®пјҢе®һйҷ…е’Ңдёә1пјҢжҳҜдёӘжӯЈж•°пјҢе’Ң15еҠ еҲ°дёҖиө·жҳҜиғҪеўһеӨ§з»қеҜ№еҖјзҡ„пјҢдёҚжҠҠе®ғеҠ иҝӣеҺ»пјҢеҸҚиҖҢдјҡеҮҸе°ҸпјҲдёҚиҝҮиҝҷйҮҢжҳҜдёҫеҮәдёҖдёӘеҸҚдҫӢпјҢдёҚзҹҘйҒ“жңүжІЎжңүж–№жі•иғҪдёҘж јзҡ„и®әиҜҒпјү

    дёҚиғҪиҝҷж ·зӣҙжҺҘйҖ’жҺЁгҖӮи§ӮеҜҹдёҠйқўзҡ„еҸҚдҫӢпјҢеҸҜд»ҘеҸ‘зҺ°пјҢд№ӢжүҖд»ҘдјҡеҮәй”ҷпјҢжҳҜжӯЈиҙҹзҡ„еҪұе“ҚгҖӮеүҚйқўзҡ„abs_max_f[i-1]зҡ„е®һйҷ…зҡ„е’Ңзҡ„жӯЈиҙҹеҸҜиғҪе’Ңnums[i]зҡ„жӯЈиҙҹдёҚдёҖж ·пјҢе°ұжңүеҸҜиғҪеҮәзҺ°дёҠйқўзҡ„жғ…еҶөгҖӮиҝҷж ·пјҢиҮӘз„¶дјҡжғіеҲ°пјҡеҰӮжһңз»ҙжҠӨдёӨдёӘж•°з»„пјҢдёҖдёӘжҳҜpos_max_fпјҢдёҖдёӘжҳҜneg_max_fпјҢе…¶з¬¬iдёӘе…ғзҙ еҲҶеҲ«иЎЁзӨәд»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„жңҖеӨ§зҡ„жӯЈзҡ„е’ҢпјҢд»ҘеҸҠжңҖе°Ҹзҡ„иҙҹзҡ„е’Ңпјӣ
    дҪҶе®һйҷ…д»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„еҸҜиғҪеҺӢж №е°ұжІЎжңүе’ҢдёәжӯЈжҲ–иҖ…е’Ңдёәиҙҹзҡ„жғ…еҶөпјҲиҝҷеҫҲе®№жҳ“дёҫеҮәеҸҚдҫӢпјүпјҢжүҖд»Ҙе…¶е®һеҸҜд»ҘжғіеҲ°пјҢжһ„йҖ max_fе’Ңmin_fеҚіеҸҜпјҢеҚіеҲҶеҲ«з»ҙжҠӨд»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„е’Ңзҡ„жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјгҖӮиҖҢжңҖз»Ҳзҡ„д»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҢеә”иҜҘе°ұжҳҜиҝҷдәҢиҖ…д№Ӣдёӯзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҲиҝҷе°ұеҘҪжҜ”пјҢжҲ‘д»¬иҰҒжүҫеҮәдёҖдёӘж•°з»„йҮҢзҡ„з»қеҜ№еҖјжңҖеӨ§зҡ„е…ғзҙ пјҢйӮЈеҸӘйңҖиҰҒжүҫеҮәиҝҷдёӘж•°з»„йҮҢзҡ„жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјпјҢеҶҚжұӮеҮәиҝҷдёӨдёӘе…ғзҙ зҡ„з»қеҜ№еҖјйҮҢжӣҙеӨ§зҡ„йӮЈдёӘеҚіеҸҜпјү
    иҰҒжһ„йҖ max_fе’Ңmin_fпјҢд»ҘеҸҠеҜ№е®ғд»¬иҝӣиЎҢйҖ’жҺЁпјҢиҝҷдёӘе°ұжҜ”иҫғз®ҖеҚ•дәҶпјҢд»ҝз…§дёҠйқўзҡ„йҖ’жҺЁејҸе°ұеҸҜд»ҘдәҶгҖӮиҝҷж¬ЎжҳҜжӯЈзЎ®зҡ„гҖӮжүҖд»Ҙд»Јз Ғе®һзҺ°еҹәжң¬е°ұеҸҜд»Ҙеҫ—еҲ°дәҶ

    д»Јз Ғпјҡ

    жіЁж„ҸдёҖдёӘе°Ҹз»ҶиҠӮпјҢдёӢйқўзҡ„д»Јз ҒйҮҢпјҢжҲ‘д»¬еҸҜд»ҘеңЁйҒҚеҺҶзҡ„ж—¶еҖҷе°ұз»ҙжҠӨжңҖз»Ҳз»“жһңпјҢиҝҷж ·е°ұдёҚйңҖиҰҒеңЁжңҖеҗҺеҶҚйҒҚеҺҶдёҖж¬Ўabs_max_fдәҶ

    1. 3.1. еӣ дёәжҳҜд»»ж„Ҹеӯҗж•°з»„пјҢеҸӘиҰҒжұӮе’Ңзҡ„з»қеҜ№еҖјжңҖеӨ§пјҢеҜјиҮҙйўҳзӣ®зңӢиө·жқҘз•ҘйҡҫгҖӮеҰӮжһңжІҝз”Ёд№ӢеүҚ[[жңҖеӨ§еӯҗж•°з»„е’Ң]]йҮҢзҡ„жҖқи·ҜпјҢе®ҡд№үдёҖдёӘabs_max_fж•°з»„пјҢе…¶з¬¬iдёӘе…ғзҙ иЎЁзӨәд»ҘnumsйҮҢзҡ„з¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҢд№ӢеҗҺе…¶е®һдёҚиғҪйҖ’жҺЁгҖӮзӣҙи§үдёҠпјҢдјјд№ҺеҸҜд»Ҙиҝҷж ·йҖ’жҺЁпјҡ\[abs\_max\_f[i]=\max(abs(abs\_max\_f[i-1]+nums[i]), abs(nums[i]))\]зңӢиө·жқҘиҝҳжҢәеҜ№зҡ„гҖӮдҪҶеҸҜд»ҘдёҫеҮәеҸҚдҫӢпјҡеҰӮжһңжңүиҝҷд№ҲдёҖдёӘж•°з»„пјҡ\([7, -1, -2, -3, 15]\)пјҢйӮЈеҪ“жҲ‘д»¬иҰҒжұӮд»Ҙ15йӮЈдёӘе…ғзҙ з»“е°ҫзҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҢз”ЁдёҠйқўйӮЈдёӘйҖ’жҺЁејҸжұӮеҮәжқҘзҡ„з»“жһңжҳҜ15гҖӮеӣ дёәжӯӨеүҚжұӮеҮәзҡ„д»Ҙ-3з»“е°ҫзҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјжҳҜ6пјҲеңЁиҝҷдёӘжғ…еҶөйҮҢпјҢеҸ–еҲ°жңҖеӨ§еҖјзҡ„жғ…еҶөжҳҜд»Һ-1еҠ еҲ°-3пјүпјҢдҪҶе®ғе®һйҷ…зҡ„е’Ңзҡ„еҖјжҳҜ-6пјҢе’Ң15еҠ еҲ°дёҖиө·дјҡеҮҸе°Ҹз»қеҜ№еҖјгҖӮеҸҚиҖҢеҰӮжһңжҠҠ7еӣҠжӢ¬иҝӣеҺ»пјҢйӮЈд№Ҳ7еҠ еҲ°-3йӮЈдёӘдҪҚзҪ®пјҢе®һйҷ…е’Ңдёә1пјҢжҳҜдёӘжӯЈж•°пјҢе’Ң15еҠ еҲ°дёҖиө·жҳҜиғҪеўһеӨ§з»қеҜ№еҖјзҡ„пјҢдёҚжҠҠе®ғеҠ иҝӣеҺ»пјҢеҸҚиҖҢдјҡеҮҸе°ҸпјҲдёҚиҝҮиҝҷйҮҢжҳҜдёҫеҮәдёҖдёӘеҸҚдҫӢпјҢдёҚзҹҘйҒ“жңүжІЎжңүж–№жі•иғҪдёҘж јзҡ„и®әиҜҒпјү
    2. 3.2. дёҚиғҪиҝҷж ·зӣҙжҺҘйҖ’жҺЁгҖӮи§ӮеҜҹдёҠйқўзҡ„еҸҚдҫӢпјҢеҸҜд»ҘеҸ‘зҺ°пјҢд№ӢжүҖд»ҘдјҡеҮәй”ҷпјҢжҳҜжӯЈиҙҹзҡ„еҪұе“ҚгҖӮеүҚйқўзҡ„abs_max_f[i-1]зҡ„е®һйҷ…зҡ„е’Ңзҡ„жӯЈиҙҹеҸҜиғҪе’Ңnums[i]зҡ„жӯЈиҙҹдёҚдёҖж ·пјҢе°ұжңүеҸҜиғҪеҮәзҺ°дёҠйқўзҡ„жғ…еҶөгҖӮиҝҷж ·пјҢиҮӘз„¶дјҡжғіеҲ°пјҡеҰӮжһңз»ҙжҠӨдёӨдёӘж•°з»„пјҢдёҖдёӘжҳҜpos_max_fпјҢдёҖдёӘжҳҜneg_max_fпјҢе…¶з¬¬iдёӘе…ғзҙ еҲҶеҲ«иЎЁзӨәд»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„жңҖеӨ§зҡ„жӯЈзҡ„е’ҢпјҢд»ҘеҸҠжңҖе°Ҹзҡ„иҙҹзҡ„е’ҢпјӣдҪҶе®һйҷ…д»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„еҸҜиғҪеҺӢж №е°ұжІЎжңүе’ҢдёәжӯЈжҲ–иҖ…е’Ңдёәиҙҹзҡ„жғ…еҶөпјҲиҝҷеҫҲе®№жҳ“дёҫеҮәеҸҚдҫӢпјүпјҢжүҖд»Ҙе…¶е®һеҸҜд»ҘжғіеҲ°пјҢжһ„йҖ max_fе’Ңmin_fеҚіеҸҜпјҢеҚіеҲҶеҲ«з»ҙжҠӨд»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„е’Ңзҡ„жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјгҖӮиҖҢжңҖз»Ҳзҡ„д»Ҙnumsз¬¬iдёӘе…ғзҙ з»“е°ҫзҡ„иҝһз»ӯеӯҗж•°з»„зҡ„е’Ңзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҢеә”иҜҘе°ұжҳҜиҝҷдәҢиҖ…д№Ӣдёӯзҡ„з»қеҜ№еҖјзҡ„жңҖеӨ§еҖјпјҲиҝҷе°ұеҘҪжҜ”пјҢжҲ‘д»¬иҰҒжүҫеҮәдёҖдёӘж•°з»„йҮҢзҡ„з»қеҜ№еҖјжңҖеӨ§зҡ„е…ғзҙ пјҢйӮЈеҸӘйңҖиҰҒжүҫеҮәиҝҷдёӘж•°з»„йҮҢзҡ„жңҖеӨ§еҖје’ҢжңҖе°ҸеҖјпјҢеҶҚжұӮеҮәиҝҷдёӨдёӘе…ғзҙ зҡ„з»қеҜ№еҖјйҮҢжӣҙеӨ§зҡ„йӮЈдёӘеҚіеҸҜпјүиҰҒжһ„йҖ max_fе’Ңmin_fпјҢд»ҘеҸҠеҜ№е®ғд»¬иҝӣиЎҢйҖ’жҺЁпјҢиҝҷдёӘе°ұжҜ”иҫғз®ҖеҚ•дәҶпјҢд»ҝз…§дёҠйқўзҡ„йҖ’жҺЁејҸе°ұеҸҜд»ҘдәҶгҖӮиҝҷж¬ЎжҳҜжӯЈзЎ®зҡ„гҖӮжүҖд»Ҙд»Јз Ғе®һзҺ°еҹәжң¬е°ұеҸҜд»Ҙеҫ—еҲ°дәҶ
