neo4j安装（Ubuntu）

Posted on 2025-11-07 In 配置相关

因为复现代码，要用到neo4j。捣鼓了好久……

首先得安装jdk，这个就类似于java的运行环境，因为neo4j是要用到java的。参考廖雪峰的jdk安装教程里面的一步，我们到oracle官网进行安装。找到对应的版本，下载压缩包就好。Ubuntu的话，就是选了这个“x64 Debian Package”的deb包。之后放到服务器上，sudo dpkg -i <deb路径>即可。此时应该能装上jdk（可能会遇到一个报错说符号链接创建错误，update-alternatives: error: error creating symbolic link '/usr/share/binfmts/jar.dpkg-tmp': No such file or directory，但无关紧要，起码在neo4j的使用上是不影响的。实在要解决好像问ai很快就可以解决，似乎只是一个小问题）。最后可以用java --version来验证是否正常安装上了，正常是应该输出版本信息的

其次是，安装neo4j。找了一些博客，看到一个这个csdn的博客，试了一下发现可以。它里面安装jdk的部分我们可以跳过，上面已经完成。安装neo4j的第一步，就是下载一个压缩包。我们可以把它的下载的链接拿过来，只需要把中间的版本改成想要的版本即可（不过注意它的链接里出现了community，也就是安装社区版。猜测要安装企业版的话，应该也是改改就好，不过目前没这个需求，就先不看了）。后面的步骤就照它的来就好
