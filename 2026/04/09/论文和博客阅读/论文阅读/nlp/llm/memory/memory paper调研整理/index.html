论文提出的方法是如何解决问题的？

论文提出了 MemoryBank，这是一个专为 LLMs 设计的长期记忆机制，由三个核心支柱组成：

   记忆存储（Memory Storage）：存储对话记录、事件摘要和用户画像。

   记忆检索（Memory Retrieval）：利用双塔密集检索模型（Dual-tower dense retrieval），根据当前对话上下文检索最相关的记忆。

   记忆更新（Memory Updating）：引入拟人化的机制来处理记忆的保留与遗忘。

论文方法的创新性

MemoryBank 的创新性主要体现在以下几个方面：

引入艾宾浩斯遗忘曲线（Ebbinghaus Forgetting Curve）：模拟人类的记忆衰减过程，根据时间流逝和记忆的重要性来选择性地遗忘或强化记忆。（现在的memory里，update、forget也是重要的功能，这篇则是初步开始把这类功能加进来。只不过它是非常naive的方法）

分层事件摘要与动态用户画像：不仅存储原始对话，还将其提炼为高层级的事件摘要，并持续更新对用户性格的理解。

极强的通用性（Versatility）：该机制可以适配闭源模型（如 ChatGPT）和开源模型（如 ChatGLM、BELLE）。 #
