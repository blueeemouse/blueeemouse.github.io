Claude code突然一直api连接不上，Request timed out

Posted on 2026-03-25

如题，很突然的问题。一开始让人怀疑是不是服务器的网络之类的出了问题，毕竟当时其它的cli工具也死活连不上。但后续其它工具正常了，就它还不行。所以又考虑是不是api那边供应出问题了（因为之前成功过，又没有改配置，所以不是api配置错了）
然而本地用这个api的Claude code还是正常的。这就很奇怪了。在网上搜了一下，也不知道什么原因，反正类似这样删掉，之后重装就行了

不过总是要删掉重装，一来是麻烦，二来是历史不是就没了吗……
