🌟 query routing and query decomposition

Hope to maintain a curated query routing and query decomposition paper repository, collecting recent high-quality research in rag or related topics. It aims to help researchers and practitioners stay updated with cutting-edge trends.

query routing

🗓️ 2025

  • Causal LLM Routing: End-to-End Regret Minimization from Observational Data (2025-12-03)(NIPS 2025)
    It proposes a unified, end-to-end LLM routing framework that minimizes expected regret directly—integrating performance estimation and routing decision into a single objective, and enables training solely from observational data without requiring costly full-feedback annotations

  • Firewall Routing: Blocking Leads to Better Hybrid Inference for LLMs (2025-11)(EMNLP 2025)
    Multi-sample pass rates + soft-label + cope with long-tail

  • Let the LLM Stick to Its Strengths: Learning to Route Economical LLM (2025-10-29)(NIPS 2025)
    Refactor llm routing as recommendation task, considering more features and extract high-order interactions

  • Router-R1: Teaching LLMs Multi-Round Routing and Aggregation via Reinforcement Learning (2025-10-24)(NIPS 2025)
    Router-R1 firstly introduce iterative, multi-round routing, using a small LLM (rl trained) as router to iteratively plan and route

  • MESS+: Dynamically Learned Inference-Time LLM Routing in Model Zoos with Service Level Guarantees (2025-10-23)(NIPS 2025)
    MESS+ formulate LLM routing as a cost-minimization problem subject to a long-term service-level agreement (SLA) constraint on response satisfaction

  • RAGRouter: Learning to Route Queries to Multiple Retrieval-Augmented Language Models (2025-10-17)(NIPS 2025)
    It generalizes standard LLM routing to RAG-aware LLM routing, conditioned on both the query and its retrieved documents, under a fixed retrieval setup

  • Cost-Aware Contrastive Routing for LLMs (2025-10-08)(NIPS 2025)
    CSCR firstly proposes to represent each model via a static, probe-based embedding instead of a learnable embedding (so can add new llms without retraining), with contrastive learning adapted to preserve cost-accuracy trade-offs

  • BEST-Route: Adaptive LLM Routing with Test-Time Optimal Compute(2025-06-28)(ICML 2025)
    Multi-head match-probability routing + test-time optimal compute

  • Learning to Route LLMs with Confidence Tokens(2025-06-19)(ICML 2025)
    Fine-tune llm to grasp when to generate two kinds of confidence token + using internal representation to construct confidence to guide routing

  • A Unified Approach to Routing and Cascading for LLMs(2025-05-22)(ICML 2025)

  • RouteLLM: Learning to Route LLMs with Preference Data(2025-02-23)(ICLR 2025)
    Chatbot-Arena human-preference bootstrapping ## 🗓️ 2024

  • RouterDC: Query-Based Router by Dual Contrastive Learning for Assembling Large Language Models(2024-09-30)(NIPS 2024)
    Dual-contrastive query-LLM embedding alignment + sample clustering for routing-assembled LLM inference

  • Query Routing for Homogeneous Tools: An Instantiation in the RAG Scenario (2024-09-30)(EMNLP 2024 findings)
    Query-performance forecasting + ILP-based batch assignment for cost-performance optimal routing of homogeneous tools in RAG

  • Adaptive-RAG: Learning to Adapt Retrieval-Augmented Large Language Models through Question Complexity (2024-05-28)(NAACL 2024)
    Proxy-labeled three-tier router for adaptive retrieval

  • Optimistic Query Routing in Clustering-based Approximate Maximum Inner Product Search (2024-05-20)(NIPS 2024)

  • Hybrid LLM: Cost-Efficient and Quality-Aware Query Routing(2024-04-22)(ICLR 2024)
    BART-scored soft-label routing (classical)

随着基于大语言模型(LLM)的生成式搜索引擎逐渐取代传统搜索（如Google的AI概览），信息生态系统的运作方式发生了根本性变化：

  • 传统搜索：用户点击搜索结果，内容提供者通过流量获得广告收入

  • 生成式搜索：AI直接生成答案，用户可能不再访问源网站，内容提供者失去流量和收入

这导致了一个关键挑战：如何设计一个激励相容的机制，确保内容提供者能根据其真实贡献获得公平补偿，从而维持健康可持续的信息生态系统。 ##### 核心研究问题

论文聚焦的具体问题是：在检索增强生成(RAG)管道中，如何高效且准确地归因计算每个检索文档对最终生成答案的贡献度。

ShaRP: SHAllow-LayeR Pruning for Video Large Language Models Acceleration，对vlm的一种剪枝加速。oppo和bupt的

ParaUni: Enhance Generation in Unified Multimodal Model with Reinforcement-driven Hierarchical Parallel Information Interaction。这个是做统一模型生成的，它关注到，视觉模型里不同层是有很丰富的特征的（低层视觉特征的到高层的语义特征），但生成模型部分没有很好地利用起来。所以论文在尝试把这部分特征利用起来，最终发现能提高生成质量。ustc和华为的

Conscious Gaze: Adaptive Attention Mechanisms for Hallucination Mitigation in Vision-Language Models。嗯，这个是研究vlm的幻觉的。它关注到，幻觉产生的一个原因是，vlm会从视觉特征漂移到语言先验（个人理解，可能类似于，看图的时候，看到一个比较典型的东西，就开始联想到自己以往见过的文字描述了，之后就抛开图像进行理解和生成了。最终体现出来就是幻觉了）。然后这篇论文希望能在vlm的内部就加上一些模块，在发现将要偏移的时候，及时制止，把注意力拉回来（感觉挺合理的，似乎也蛮有意思的。只不过是中国矿业大学的，稍微的让人有点担心（狗眼看人低了……）

Probing the effectiveness of World Models for Spatial Reasoning through Test-time Scaling研究世界模型的。可以开拓下眼界吧。它关注的问题似乎是，vlm在处理需要多视角理解和具身视角转换的空间推理任务上，表现不够好。现有方法在测试的时候，使用世界模型，来缓解这一问题，并使用启发式验证器选择有用的视角，但是对这些验证器缺乏系统的行为分析（大概是说，一直在用验证器解决多视角问题，但是对这个验证器不够了解是吗？）

HQ-DM: Single Hadamard Transformation-Based Quantization-Aware Training for Low-Bit Diffusion Models。这篇关注的是扩散模型的量化。简单讲，扩散模型在量化的时候会遇到困难，似乎是因为它的激活矩阵里面会有异常值，所以用一般的量化方法性能会显著下降。论文正是针对这一点，进行处理。ustc的

USV: Unified Sparsification for Accelerating Video Diffusion Models。也是做扩散模型加速的。它的出发点是，现有方法对扩散模型加速的时候，一般都是针对一个瓶颈去优化的，容易遇到收益递减问题。所以论文提出从多个角度下手同时优化（国科大和腾讯混元的，看着还蛮权威的）

Bring Your Dreams to Life: Continual Text-to-Video Customization。这篇有点类似于定制化的多模态模型，不过它聚焦的是定制视频生成。它的出发点是，在定制的文本到视频生成任务中，个性化概念都是不变的（也就是不能更新概念吧？），而且学新的概念的时候，会发生遗忘（经典）（好多人做的工作）
